# Plotting Style Guide

## General
- Use matplotlib with dpi=120
- Font size: 11pt default, 18pt for suptitle
- Grid: alpha=0.3
- Always label axes

## Step Functions
- Plot on [-1, 1] interval using `plt.step(..., where='post')`
- Add horizontal dashed line at y=0.5 (lightgray)
- Show upper bound value in title with 6 decimal places
